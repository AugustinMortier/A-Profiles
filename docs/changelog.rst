0.2.1
^^^^^^^
Nov 29, 2021

- Add CLI for facilitating deployment on ecFlow 

e.g:
    - ``./run/aprorun.py --date 2021-09-09``
    - ``./run/aprorun.py --from 2021-09-09 --to 2021-09-10``
    - ``./run/aprorun.py --today``
    - ``./run/aprorun.py --today --yesterday``
